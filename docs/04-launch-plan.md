# Bluff Arena 上线执行清单

## A. 已完成（本地可玩）

- 游戏化桌面交互（HUD、牌桌、手牌区、动作条）
- WebSocket 实时同步
- 对局闭环（开局、出牌、质疑、聊天、结算、再来一局）
- 基础容器化（Dockerfile）

## B. 上线前必须补齐

1. 多模型接入
- 抽象 `LLMAdapter` 接口
- 接入 OpenAI/Anthropic/Google 三适配器
- 加超时与降级策略

2. 数据与状态
- PostgreSQL 持久化对局
- Redis 存储在线房间状态和短期事件
- 重连恢复与断线保护

3. 安全合规
- 账号体系（匿名游客 + 第三方登录）
- 速率限制、IP 风控、反刷策略
- 内容审核服务化（文本、昵称、房间名）

4. 可靠性
- 统一日志 + 指标 + tracing
- 错误告警通道
- 压测与容量评估

5. 发布系统
- CI/CD（测试、构建、镜像发布）
- 生产环境配置管理
- 灰度发布与回滚方案
