# Personal Skill Stack

## Current Loading Path

- Codex-visible skill directory: `~/.codex/skills`
- Impeccable installer output: `~/.agents/skills`
- Action taken: symlink only the most useful Impeccable skills into `~/.codex/skills`

## Imported Impeccable Skills

- `frontend-design`
- `polish`
- `audit`
- `critique`

These cover the highest-value frontend quality loop:

1. choose a strong visual direction
2. implement with taste
3. polish details
4. audit quality

## Custom Skills Added

### `figma-ship-page`

Use for Figma URL or node to production page work.

Flow:
- `figma`
- `frontend-design`
- `polish`
- `audit`

### `landing-page-polish`

Use when a landing page already exists and needs to become more convincing.

Flow:
- `critique`
- `frontend-design`
- `polish`
- `audit`

### `content-launch-kit`

Use when one request spans research, page copy, image direction, and Xiaohongshu preparation.

Flow:
- `notion-research-documentation` when needed
- `imagegen` when needed
- `xhs-publisher` when needed

## Recommended Use

- Figma to code: ask for `figma-ship-page`
- Existing marketing page upgrade: ask for `landing-page-polish`
- Product launch asset pack: ask for `content-launch-kit`

## Why This Structure

- Small enough to stay reliable
- Centered on your real tasks instead of generic skill collecting
- Keeps Codex-facing skills in one directory it can actually load
